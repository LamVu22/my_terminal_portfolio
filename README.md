# Terminal Portfolio by Lam Vu

![ts](https://badgen.net/badge/Built%20With/TypeScript/blue?style=flat-square)

My portfolio website in terminal style, built with React, TypeScript, and Styled-Components. Supports multiple themes and keyboard shortcuts.

## Features

- Responsive Design
- Multiple themes (type `themes` to see options)
- Autocomplete (TAB | Ctrl + i)
- Command history navigation (Up/Down arrows)
- View command history

## Tech Stack

**Frontend** - [React](https://reactjs.org/), [TypeScript](https://www.typescriptlang.org/)
**Styling** - [Styled-Components](https://styled-components.com/)
**State Management** - [ContextAPI](https://reactjs.org/docs/context.html)
**Testing** - [Vitest](https://vitest.dev/), [React Testing Library](https://testing-library.com/)

## Running Locally

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
